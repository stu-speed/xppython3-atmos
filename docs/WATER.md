# WATER SUBSYSTEM  

Wind‑Sea + Swell + Water Physics + Floatplane Interaction  

The Water subsystem synthesizes the complete state of the water surface at the aircraft’s location.  
It combines **local wind‑sea**, **global swell**, **wave physics**, **water drag**, and **wave normals**.  
Outputs drive floatplane physics, rendering, and downstream haptic systems.

The Water subsystem does **not** modify wind.

# 2. WAVE MODEL

Two‑component spectrum:


\[
E(f,\theta) = E_{ws}(f,\theta) + E_{sw}(f,\theta)
\]



## 2.1 Wind‑Sea (Local)


\[
f_{p,ws} = \frac{g}{2\pi U_{10}}
\]




\[
D_{ws}(\theta) = \cos^{s}(\theta - \theta_{wind})
\]




\[
H_{ws} = a U_{10}^{b} (1 - e^{-c \cdot fetch})
\]




\[
C = k U_{10}
\]



## 2.2 Swell (Global)


\[
f_{p,sw} = \frac{1}{T_{swell}}
\]




\[
D_{sw}(\theta) = \cos^{n}(\theta - \theta_{swell})
\]




\[
A_{sw}(t) = A_0 e^{-kt}
\]



---

# 3. DERIVED PARAMETERS



\[
H_s = 4\sqrt{\int E(f) df}
\]




\[
T_p = \frac{1}{f_p}
\]




\[
\theta_p = \arg\max_\theta \int E(f,\theta) df
\]




\[
S = \frac{H_s}{\lambda_p}
\]



---

# 4. WAVE STATE (TIME‑EVOLVING)



\[
\phi_i(t) = \phi_i(0) + \omega_i t
\]




\[
\eta(x,y,t) = \sum_i A_i \cos(k_i x \cos\theta_i + k_i y \sin\theta_i - \omega_i t + \phi_i)
\]




\[
n_x = -\frac{\partial \eta}{\partial x},\quad
n_y = -\frac{\partial \eta}{\partial y},\quad
n_z = \sqrt{1 - n_x^2 - n_y^2}
\]



---

# 5. WATER DRAG



\[
D = d_0 + d_1 H_s + d_2 C + d_3 S
\]




\[
\nu = \nu_0 e^{-k_{temp}(T - T_0)}
\]

# 8. NOTES

- Lakes use **wind‑sea only**.  
- Oceans use **wind‑sea + swell**.  
- Wave state is time‑evolving and sampled each frame.  
- Floatplane physics consumes wave normals, drag, and displacement.
